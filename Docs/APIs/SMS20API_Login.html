Login

Login into SMS 2.0 platform.

HBufC8* Login(const TDesC8& aLogin, const TDesC8& aPwd)

Parameters
aLogin
  String with a telephone number active in SMS 2.0 platform

aPwd
  String with the password of the telephone in SMS 2.0 platform

Return
A String with the Session ID if connected, else an empty string.
