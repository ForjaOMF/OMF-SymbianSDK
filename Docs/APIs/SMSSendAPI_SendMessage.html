SendMessage

Make an HTTP transaction to send a SMS.

TBuf8<1024> SendMessage(const TDesC8& aLogin, const TDesC8& aPwd, const TDesC8& aDest, const TDesC8& aMsg);

Parameters
aLogin
  String with sender's telephone number

aPwd
  String with sender's password

aDest
  String with destination telephone number

aMsg
  String with message text

Return
If the message is correctly sent, OK

else
  • 1 - Incorrect user
  • 4 - Service unavaliable
  • 5 - Incorrect password
  • 11 - Incorrect destination
  • 12 - Destinatation not specified
