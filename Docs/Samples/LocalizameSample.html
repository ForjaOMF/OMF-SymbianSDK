Localizame

A sample application that allows you to locate an user of Localizame service through an HTTP interface. It demonstrates:
  • Localizame API
Download Sample


In the src/ClientAppUI.cpp you would find the following lines than are the parameters for the Login in Localizame example, you can change the parameters at your will.

// loggin in Localizame platform
_LIT8(klogin8, "655522555");
_LIT8(kpsw8, "password");
_LIT8(kAutorice, "655522555");
_LIT8(kLocate, "655522555");
_LIT8(kUnautorice, "655522555");

iClient->Login(klogin8, kpsw8);
iClient->Authorize(kAutorice);

TBuf8<1024> aResponse;
aResponse = iClient->Locate(kLocate);

TRAPD(err, iAppView->AddToOutputWindowL(aResponse));
if(err)
Panic(EClientView);
iClient->Unauthorize(kUnautorice);
iClient->Logout();
