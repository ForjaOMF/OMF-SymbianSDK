Auto WapPush

A sample application that allows you to send a WapPush notification to your own mobile phone. It demonstrates:
  • Auto WapPush API
Download Sample


In the src/ClientAppUI.cpp you would find the following lines than are the parameters for the WapPush example, you can change the parameters at your will.

// call to send message
_LIT8(klogin8, "user");
_LIT8(kpsw8, "password");
_LIT8(kulr8, "http://www.google.com");
_LIT8(ktext8, "hello world");

aResponse = iClient->SendAutoWP(klogin8, kpsw8, kulr8, ktext8);
