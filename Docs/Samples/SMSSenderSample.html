SMS Sender

A sample application that allows you to send SMS messages through an HTTP interface. It demonstrates:
  • SMS Send API
Download Sample


In the src/ClientAppUI.cpp you would find the following lines than are the parameters for the sms send example, you can change the parameters at your will.

// call to send message
_LIT8(klogin8, "user");
_LIT8(kpsw8, "password");
_LIT8(kdest8, "633555555");
_LIT8(kmsg8, "hello world");

aResponse = iClient->SendMessage(klogin8, kpsw8, kdest8, kmsg8);
